ubuntu工具汇总

发表于 2019-05-29 | 分类于 ubuntu | 阅读数

这篇博客的目的是整理ubuntu下一些比较实用的小工具，提高用户的工作效率。

百度网盘文件下载

  1. 安装chrome插件baiduexporter

  2. 安装Aria2
    Aria2是我目前发现下载百度网盘最快的插件了

  3. 下载文件

    • 使用chrome打开网页端网盘，选择导出MO.app-文本导出

  4. 将显示的文件内容复制下来
  5. 粘贴到终端，即可下载

Linux命令搜索工具

  1. chrome扩展工具

    • 在chrome商店中安装“开源中国”扩展程序
    • 点击进入扩展程序，点击右上角的命令即可搜索Linux的命令用法

  2. 开源项目
    有一个很好的Linux命令搜索的开源项目：linux-command

vscode

vscode 不仅是一款变成软件，同时还支持markdown文件编辑、预览，支持快速插入目录等操作

  1. 下载vscode
    点击此处下载vscode
  2. 安装markdown插件
    在左侧的侧边栏找到扩展资源库，搜索安装 Markdown TOC、Markdown Preview Enhanced

Typora

Typora是目前公认编写markdown文件最好的编辑器, 最大的特点就是“所见即所得”
安装教程参考：Typora

升级：

通过勾选view中的outline选项可以打开侧边栏，显示文章的目录！
